# Java Spring Course — Tài Liệu Kiến Thức

Tổng hợp đầy đủ kiến thức Java và Spring Framework, từ cơ bản nhất đến nâng cao.
Dùng làm nguồn nội dung cho app học tập — mỗi file là một bài học hoặc nhóm keywords.

## Cấu Trúc

```
doc/
├── 01-java-core/          # Java Core Fundamentals (Beginner)
├── 02-java-oop/            # Java OOP (Beginner-Intermediate)
├── 03-java-advanced/       # Java Advanced (Intermediate)
├── 04-spring-core/         # Spring Framework Basics (Intermediate)
├── 05-spring-boot/         # Spring Boot (Intermediate-Advanced)
├── 06-spring-data/         # Spring Data & Persistence (Advanced)
├── 07-spring-security/     # Spring Security (Advanced)
├── 08-spring-cloud/        # Spring Cloud & Microservices (Advanced-Expert)
└── 09-real-world/          # Real-world Project Patterns (Expert)
```

## Quy Ước Nội Dung

- Mỗi file `.md` là một bài học hoặc nhóm keywords
- Mỗi bài bao gồm: giải thích lý thuyết, keywords kèm định nghĩa, code examples
- Keywords được đánh dấu bằng heading `## Keyword: <tên>`
- Code examples nằm trong code blocks có syntax highlighting Java
- Nội dung viết bằng tiếng Việt, thuật ngữ kỹ thuật giữ nguyên tiếng Anh
