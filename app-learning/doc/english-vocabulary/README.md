## 🎯 Cách Sử Dụng

1. **Đọc theo module**: Mỗi file tương ứng với một nhóm chủ đề trong khóa học Java/Spring
2. **Tra cứu nhanh**: Dùng Ctrl+F để tìm từ cần tra
3. **Luyện phát âm**: Mỗi từ có phiên âm IPA để luyện đọc đúng
4. **Học qua ngữ cảnh**: Mỗi từ có 2-3 câu ví dụ thực tế trong công việc lập trình
5. **Mở rộng vốn từ**: Phần "Từ liên quan" giúp liên kết kiến thức

## 📝 Format Mỗi Mục Từ Vựng

```markdown
### Tên Từ/Cụm Từ
- **Phát âm:** /phiên âm IPA/
- **Loại từ:** Noun / Verb / Adjective / ...
- **Nghĩa tiếng Việt:** Giải thích ngắn gọn bằng tiếng Việt
- **English definition:** Clear, concise definition in English
- **Cách dùng trong thực tế:**
  1. Câu ví dụ 1 (Dịch tiếng Việt)
  2. Câu ví dụ 2 (Dịch tiếng Việt)
  3. Câu ví dụ 3 (Dịch tiếng Việt)
- **Từ liên quan:** term1, term2, term3
```
